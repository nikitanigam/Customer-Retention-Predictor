# Customer Retention Prediction System

## Overview

This project is a Machine Learning-based Customer Retention Prediction System developed using Python, Scikit-Learn, and Streamlit.

The application predicts whether a telecom customer is likely to churn based on customer demographics, account information, and subscribed services.

## Features

* Predict customer churn risk
* Interactive Streamlit web application
* Customer information input form
* Real-time prediction results
* Customer summary dashboard

## Dataset

The project uses the IBM Telco Customer Churn Dataset containing customer demographics, service subscriptions, billing information, and churn status.

## Model Performance

* Model: Logistic Regression
* Accuracy: 80.3%

## Tech Stack
